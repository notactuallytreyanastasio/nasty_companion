Nasty Clone

A retro-styled real-time bookmark feed with multiple views

Features

  • Real-time bookmark updates via Phoenix Channels
  • Multiple retro-themed views (Terminal & Feed)
  • Tag-based filtering with Python client
  • Dot-matrix printer aesthetic

Views

/bookmarks

PDP-11 style terminal view with multi-column layout

/feed

Dot-matrix printer style continuous feed

Tech Stack

  • Phoenix LiveView
  • Phoenix Channels
  • Tailwind CSS
  • Python WebSocket Client
